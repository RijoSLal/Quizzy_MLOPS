# Quizzy MLOps 🗘

This repository contains all notebook experiments, training data, and pipeline configurations for the face emotion recognition model used in [Quizzy](https://github.com/RijoSLal/Quizzy).

## 📊 Data & Experiment Tracking
We use **DagsHub** to version our datasets (via DVC) and track model training metrics. 
* 🔗 **View the DagsHub Project:** [DagsHub Repository Link](https://dagshub.com/slalrijo2005/Quizzy_MLOPS)

## 📁 Repository Structure
* `emotion_detection_model.ipynb` — Exploratory data analysis, model prototyping, and training experiments.
* `affectnet.dvc` — Data tracking files (e.g., `.dvc` pointers to our training sets).
